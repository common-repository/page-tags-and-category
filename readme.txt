This plugin adds category and tags functionality for WordPress pages. WordPress by default do not have this functionality. Also from settings, one can choose to enable only tags for pages or only category for pages or both.

== Description ==
This plugins enables the categories metabox and the tags metabox in the New or Edit Page interface.

The plugin has minimum settings to enable only tags for pages or only categories for pages or both tags and categories for pages from its setting page.

**Note:** This plugin will not add any display of page categories or page tags to your template files.

Features:

1. Enables Tags For Pages
2. Enables Category For Pages

**Plugin's Website: [https://digitaladquest.com/wordpress-plugins/](https://digitaladquest.com/wordpress-plugins/)**

**Author's Website: [https://digitaladquest.com/](https://digitaladquest.com/)**


== Installation ==
1. Extract all files from the ZIP file, and then upload the plugin's folder to /wp-content/plugins/.
2. Activate the plugin through the \"Plugins\" menu in WordPress.
3. Go to the plugin settings and enable or disable the options.

== Frequently Asked Questions ==
= Will this plugin add display of categories or tags to template files? =
This plugin will not add any display of page categories or page tags to your template files.
